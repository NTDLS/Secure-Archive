HardKey - How to Use



Using a HardKey is an advanced feature. You should read and understand the Encryption Walk through and Decryption Walk through before continuing, as this section builds on the information obtained there.

Encrypting with a HardKey

Overview:

You can use your HardKey at the New Password windows to create new encrypted packages.

Steps:
  1. Get to the point where you can specify a new password in the archive creation process. As discussed in the Encryption Walk through.
  2. Type in the password that you used when creating the HardKey file in both the "Password" and "Confirm Password" text boxes.
  3. Check the "Use HardKey file" check box. (figure 1)
  4. Find and select the hard key file that you created previously.
  5. Click Encrypt.

Decrypting with a HardKey
Overview:

You can use your HardKey at the Open Password windows decrypt / view encrypted packages.

Steps:
  1. Get to the point where you can enter the password in the archive decryption process. As discussed in the Decryption Walk through.
  2. Type in the password that you used when creating the HardKey file in the "Password" text box.
  3. Check the "Use HardKey file" check box. (figure 2)
  4. Find and select the hard key file that you created previously.
  5. Click OK.
