HardKey - How to Create



To create a HardKey, you will need to use the "Secure Archive HardKey Generator" which can be found in the start menu group where Secure Archive was installed. (e.g. Start->Programs->NetworkDLS->Secure Archive->HardKey Generator)

Read below for instructions...

The Secure Archive HardKey Generator

Creation Steps:
  1. Create a new password and type it into both the "Password" and "Confirm Password" text boxes. Remember this password, if you lose it - your data will be unrecoverable.
  2. Select a key size by dragging the slider to a desired length.
  3. Click OK.
  4. Select the desired location to save the generated key file to. We recommend saving it directly to removable media, such as a USB thumb drive. Keep this file, if you lose it - your data will be unrecoverable.
