HardKey - Overview



What is a "HardKey"?
A HardKey is nothing more than a very long random password that is generated by Secure Archive and stored in a file.

Each HardKey file is itself encrypted, and has an accompanying (user specified) password which must be used in conjunction with the HardKey file. In this way, the HardKey file complements your password by adding a new level of access protection.
