Secure Delete - How to Use



Securely deleting files and folders with Secure Archive is very easy.

Steps:

  1. Open Secure Delete by going to the Start menu and clicking on NetworkDLS -> Secure Archive -> Secure Delete.
  2. You should now see the Secure Delete Wizard window (figure 1).
  3. Using your mouse, Drag-and-drop any files that you are sure you want to remove permanently to the secure delete wizard.
  4. Click delete. (figure 3)..
