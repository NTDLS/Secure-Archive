Encrypting - Walkthrough



There are two methods of encrypting files and/or folders with Secure Archive, these are discussed below.

Method 1 - On-the-fly:

Overview:

The on-the-fly method is probably the quickest, simplest and most productive method of encrypting your files and folders.

Steps:
  1. Find files or folder that you would like to encrypt in windows explorer or on your desktop.
  2. Click the file or folder with the right mouse button.
  3. Navigate to the pop-up menu sub-item called "Secure Archive" and select "Encrypt-to..." (figure 1).
  4. At this point you will be able to specify a password and encrypt your data (figure 2).

Method 2 - Through the Secure Archive Interface:
Overview:

Adding files using the Secure Archive interface is one of the easiest ways to add files to a pre-existing encrypted package file.

To add files to a new non-existing encrypted package:
  1. Open Secure Archive by clicking the windows Start button the going to: Programs -> NetworDLS -> Secure Archive -> Secure Archive
  2. Drag files from windows explorer or from the desktop to the file list on the main interface windows (figure 3) - or - click "Add" on the tool bar and select the you want to add.
  3. You will be asked if you would like to create a new archive. Click Yes.
  4. At this point you will be able to specify a password and encrypt your data (figure 4).
To add files to a pre-existing encrypted package:
  1. Open the existing package by double clicking on it in windows explorer or by using the "Open" tool bar button on the main interface window.
  2. Drag files from windows explorer or from the desktop to the file list on the main interface windows - or - click "Add" on the tool bar and select the you want to add.
If you attempt to add files without first opening an existing archive, you will be prompted for a password to encrypt the data with - otherwise the password used on the existing archive will automatically be used on the newly added files.
