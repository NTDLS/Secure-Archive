Encrypting - Walkthrough



There are a few advanced options you can utilize when encrypting files, these options are discussed below.



Advanced Encryption Options:


  1. Save File Time stamps: Will save the original files creation & last modified times and restore them when the files are decrypted.
  2. Save file attributes: Will save the original files attributes (hidden, read-only, ect.) and will restore these attributes when the files are decrypted.
  3. Calculate file CRCs: This option enables CRC32 calculations. This option will allow you to verify the encrypted files integrity either at decryption or by using the test archive option.
  4. Save file paths: This option will save the file paths of the encrypted files and will allow you to extract the files to their original directory structure.
  5. Include hidden / system files: Allows you to either include or ignore hidden & system files.
  6. Securely delete source files: Will cause the original source files to be deleted using n-number of passes upon successful encryption.
  7. Save as default: If this option is checked, the options above will be saved as default. Otherwise, the selected options are for this encryption "session" only.
