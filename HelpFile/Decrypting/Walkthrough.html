Decrypting - Walkthrough



Like encrypting, there are two methods of decrypting files and/or folders with Secure Archive, these are discussed below.

Method 1 - On-the-fly:

Overview:

The on-the-fly method is probably the quickest, simplest and most productive method of decrypting an entire archive.

Steps:
  1. Find an existing encrypted package within windows explorer.
  2. Right click on the archive and find the "Secure Archive" pop-up sub-menu item.
  3. Select "Decrypt to..." (figure 1).
  4. At this point you will be prompted for the correct password if one was used to encrypt the archive (figure 2).
  5. Select the location on your computer where you would like the files decrypted to (figure 3).
Notes:
  1. If you selected "Decrypt to..." as mentioned in step 3 above, you will be prompted for a location to decrypt the files to.
  2. If you selected "Decrypt to <Specific Location>" as opposed to step 3 above, you will not be prompted for an extraction location.

Method 2 - Through the Secure Archive Interface:
Overview:

Decrypting files using the Secure Archive interface is one of the easiest ways to search and decrypt specific files.

Steps:
  1. Open an existing encrypted package by double clicking it or by browsing for it using the "Open" button on the main interface tool-bar.
  2. Find the files in the Secure Archive file list that you would like to decrypt.
  3. Right click on the files you want to decrypt and click, "Extract Files..." (figure 4).
  4. At this point you will be able to specify the destination location for your decrypted files.(figure 5).
Notes:
  1. Instead of clicking extract (as mentioned in step 3 above) you can simply drag the files to a desired destination in windows explorer.
  2. You can select multiple files by holding down the SHIFT or CTRL key. The SHIFT key allows you to select a range of files, while holding down the CTRL key allows you to select multiple individual files.
