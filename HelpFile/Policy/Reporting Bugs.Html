Reporting Bugs



This application has been tested on Windows 2000, XP, Vista & 7.

The windows NT, 95 & 98 kernel does not have sufficient threading support to run this application, therefore it will not run on these platforms.

In past releases, few bugs have been reported but all found have been fixed. If this application crashes or gives you an error, please take note of any error messages and if possible also note the circumstances in which it occurred, then visit the bug reporting page on NetworkDLS's website to submit any collected error information.
