Configuration - Look & Feel



This tab allows you to change basic functionality and the look / feel of security to meet your preferences.

See below the image for a detailed break down of options.



Enable grid icons:

  • This option will turn the grid icons on/off. Disabling grid icons can significantly boots performance and reduce program requirements when working with archives with many files.

Show grid lines:
  • This option will turn the grid icons on/off. Disabling grid icons can boots performance and reduce program requirements when working with archives with many files.

Enable "full row select":
  • This option will turn grid "Full row select" on/off. Some people feel this can make drag-drop easier.

Enable drag-n-drop:
  • This option allows you turn off drag drop.

Always run with low priority:
  • This option allows you to always run Secure Archive in low thread priority. This will free resources for other programs, but can also significantly increase the amount of time it takes for Secure Archive to perform if other intensive applications are running.

Warn about blank passwords:
  • This allows to to turn the "blank password warning" window on/off.

Sorting Order:
  • Order: Allows you to specify what order files will be in when Secure Archive opens.
  • Direction: Allows whether the files will be sorted ascending or descending by the sort column.

Keep selected grid items visible while sorting:
  • This option causes Secure Archive to reselect and scroll the top one of selected rows prior to a reorder operation.


You can also Change Password Strength Alerts on this tab.
