Configuration - Processor Settings



This tab gives you a detailed list of your installed processors and their capabilities. You can use this setup tab to tell Secure Archive which processors it is allowed to use.

See below the image for a detailed break down of options.



Processors available to SecureArchive:

  • Tells Secure Archive which processors it is allowed to use.
  • All processors are selected by default.
Enable Thread Priority Boost:
  • Enables Secure Archive's ability to temporarily boost the priority of any of its threads.
  • This option is enabled by default.
Enable Process Priority Boost:
  • Enables Secure Archive's ability to temporarily boost the priority of its worker threads.
  • This option is enabled by default.
